# Locked Out

A browser extension that helps you manage your screen time by blocking distracting websites.

## Screenshot

![Locked Out Screenshot](./Screenshot%202026-02-07%20at%205.24.19%20AM.png)

## Features

- Site blocking with customizable blocklist
- "Super blocklist" for sites you want to completely block
- Time tracking for blocked sites
- Permission-based allowing (choose to allow a site for a session)

## Installation

1. Download or clone this repository
2. Open Chrome/Edge and go to `chrome://extensions`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select this folder

## Configuration

1. Click the extension icon to open settings
2. Add sites to your blocklist
3. Set up your API key in the options page
